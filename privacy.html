개인정보 처리방침

Elliesbang Image Editor(이하 “서비스”)는 사용자의 개인정보를 소중히 다루며, 다음과 같은 원칙에 따라 처리합니다.

1. 수집하는 정보

서비스는 기본적으로 브라우저 내에서 모든 처리를 수행하며, 서버에 이미지를 업로드하지 않습니다. 로그인 시에는 이메일 주소와 인증 코드만을 사용하며, GitHub Pages 환경에서는 로컬 스토리지에만 보관됩니다.

2. 정보의 이용

저장된 세션 정보는 편집 크레딧 계산 및 관리자 배너 노출을 위해 사용됩니다. Netlify + Hono 배포 시에는 OTP 인증 및 관리 기능을 위해 이메일이 서버에 전송될 수 있습니다.

3. 보관 및 파기

브라우저 로컬 스토리지에 저장된 정보는 사용자가 로그아웃하거나 브라우저 데이터를 삭제하면 즉시 파기됩니다.

4. 문의

개인정보 관련 문의는 privacy@elliesbang.com으로 연락해 주세요.
