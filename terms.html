이용약관

이 이용약관은 Elliesbang Image Editor(이하 “서비스”)의 이용과 관련하여 필요한 권리, 의무 및 책임 사항을 규정합니다.

1. 서비스 제공

서비스는 HTML5 Canvas 기반 이미지 편집 도구 및 관리자 도구를 제공합니다. GitHub Pages 배포 시에는 모든 기능이 데모 모드로 동작합니다.

2. 사용자 책임

사용자는 본 서비스에 업로드하는 콘텐츠에 대한 권리를 보유해야 하며, 저작권 등 타인의 권리를 침해해서는 안 됩니다.

3. 보증의 부인

서비스는 “있는 그대로” 제공되며, 데이터 손실에 대한 책임을 지지 않습니다. Netlify + Hono 환경에서 제공되는 API 연동 시에도 안정적인 사용을 위해 백업을 권장합니다.

4. 약관 변경

서비스는 필요한 경우 약관을 변경할 수 있으며, 변경 사항은 본 페이지를 통해 공지됩니다.
